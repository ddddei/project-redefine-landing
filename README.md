# 프로젝트 리디파인 랜딩페이지

고립·은둔 청년, 쉬었음 청년, 관계 단절감을 느끼는 직장인 청년을 위한 **프로젝트 리디파인** 소개 랜딩페이지입니다.

## 기술 스택

- Vite
- React
- TypeScript
- Tailwind CSS

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 안내되는 로컬 주소(기본값 `http://localhost:5173`)로 접속하세요.

## 빌드

```bash
npm run build
npm run preview
```

## 배포

이 프로젝트는 정적 빌드 결과물(`dist`)을 생성하므로 Vercel에 바로 배포할 수 있습니다.

- Build Command: `npm run build`
- Output Directory: `dist`
